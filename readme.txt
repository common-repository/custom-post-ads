== Description ==

Welcome to the Custom post ads design your custom ads in a few minutes.

This will help you customer to reach new pages or sites.

You can easily create you custom ad.

In this Custom post ads you can easily set your post to a perticular category.


= Features =

- Easy To use
- Add Action
- Make Active or Inactive
- Toggle Page or post
- Change Call to Action
- Add Image
- Create Sponsored section
- Select Post Category
- Visible before, after or both
- Customize width & height
